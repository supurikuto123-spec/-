使い方ガイド

Sutemeadoは、登録不要・パスワードでいつでもアクセスできるシンプルな一時メールサービスです。このガイドでは、初めての方でも簡単に使えるよう、ステップバイステップで使い方を説明します。

1
サイトにアクセスする

ブラウザ（Chrome、Safari、Edgeなど）で sutemeado.com にアクセスしてください。アクセスすると、自動的に新しい一時メールアドレスが生成されます。

ヒント： スマートフォン・タブレット・PCのどれからでも利用可能です。お好きな端末でご利用ください。
2
メールアドレスをコピーする

画面上部に表示されている「あなたのアドレス」（例：xxxxxx@sutemeado.com）をコピーします。コピーボタン（📋アイコン）をクリックすると簡単にコピーできます。

重要： パスワードも必ず保存してください！同じアドレスに再度アクセスするには、メールアドレスとパスワードの両方が必要です。
3
メールアドレスを入力する

試したいサービスやフォームの「メールアドレス入力欄」に、先ほどコピーしたアドレスを貼り付けます。確認メールや認証コードを受信したいサービスで、このアドレスを使ってください。

4
メールを待つ

メールの送信元からメールが届くまで、数秒〜数分かかる場合があります。Sutemeadoでは受信ボックスが約5秒ごとに自動更新されるため、画面を開いたまま待っていれば、自動的にメールが表示されます。

急ぐ場合： 「更新」ボタンをクリックすると、即座に最新の状態を確認できます。
5
メールを開く

受信ボックスに届いたメールをクリックすると、本文が表示されます。認証コードやリンクを確認し、必要な操作を行ってください。

注意： 一時メールは長期保存向けではありません。重要な情報は必ずコピーまたはスクリーンショットで保存してください。

便利な機能

パスワード保護
各メールアドレスには固有のパスワードが設定されます。ブラウザを変えても、メールアドレスとパスワードがあればどこからでもアクセスできます。
自動更新
受信ボックスは約5秒ごとに自動更新されます。手動でリロードする必要なく、新着メールをリアルタイムに確認できます。
ワンクリックコピー
メールアドレスとパスワードは、コピーボタンをワンクリックで簡単にコピーできます。手入力の間違いを防げます。
パスワード変更
いつでもパスワードを変更できます。より強固なパスワードに変更して、セキュリティを向上させることも可能です。
メール削除
不要なメールは個別に削除可能です。また、「全メール削除」機能で受信ボックスを一度に空にすることもできます。
新規アドレス作成
いつでも新しいメールアドレスを作成できます。古いアドレスを削除せずに、新しいアドレスを追加することも可能です。

APIの活用方法

Sutemeadoは開発者・テスター向けにAPIを提供しています。プログラムから一時メールアドレスを操作することで、自動化テストや開発作業を効率化できます。

// 新規アドレスを作成 curl https://sutemeado.com/api/new-address // ログインしてメール一覧を取得 curl -X POST https://sutemeado.com/api/login \ -H "Content-Type: application/json" \ -d '{"address":"xxx@sutemeado.com","password":"xxx"}'

詳細はトップページの「API」メニューからご確認ください。

メニューの使い方

画面右上のハンバーガーメニュー（三本線のアイコン）をクリックすると、各種機能にアクセスできます：

  • ログイン： 既存のアドレスにパスワードでアクセス
  • 新しいメールアドレスを作成： 新規アドレスを発行
  • 全メール削除： 受信ボックスを空にする
  • アドレスを削除： このアドレス自体を削除
  • APIドキュメント： APIの詳細仕様を確認
  • 設定： テーマ（ライト/ダーク）の切り替え
  • 稼働状況： サービスの稼働状況を確認
  • お知らせ： 最新のニュースとお知らせ
  • よくある質問： 詳細なFAQを確認

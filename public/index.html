Sutemeado（ステメアド）とは？

Sutemeadoは、登録不要ですぐに使える一時メール（使い捨てメール・捨てメアド）サービスです。ブラウザにアクセスするだけで、ランダムなメールアドレス（例：xxxxxx@sutemeado.com）が自動生成され、すぐにメールを受信できます。

迷惑メールやスパムメールを避けたいとき、試したいサービスにメアドを登録したいけど本アドレスは教えたくないとき、開発テスト用にメール受信が必要なときなど、さまざまな場面でご活用いただけます。

登録不要
メールアドレス不要・個人情報不要
即座に使用
アクセスしてすぐメアド発行
完全無料
月額料金・課金なし

3ステップで簡単に使える

STEP 1: アクセス

ブラウザで sutemeado.com にアクセス。自動的に一時メールアドレスが生成されます。

STEP 2: コピー

表示されたメールアドレスをコピー。パスワードも必ず保存してください（後でログインに必要です）。

STEP 3: 受信

試したいサービスのメールアドレス欄に貼り付け。届いたメールは画面下部で確認できます（約5秒自動更新）。

安全性とプライバシー保護

HTTPS暗号化対応

ブラウザとサーバー間の通信はTLS暗号化で保護され、第三者からの傍受を防ぎます。

パスワード保護

各メールアドレスには固有のパスワードが設定され、ブラウザを変えてもログインできます。

個人情報不要

名前、電話番号、住所など一切不要。ブラウザでアクセスするだけですぐに使えます。

注意：一時メールは短期間で削除されるため、銀行口座や重要なSNSアカウントの登録には使用しないでください。詳しくはFAQをご確認ください。

よくある質問

Q. 一時メールとは何ですか？
一時メール（使い捨てメール・捨てメアド）は、一定期間だけ利用して破棄できるメールアドレスです。Sutemeadoでは、ブラウザでサイトにアクセスするだけで自動的に生成され、迷惑メール対策やプライバシー保護に役立ちます。
Q. 無料で使えますか？
はい、Sutemeadoは完全無料で提供しています。登録料や月額料金などは一切不要で、広告収入で運営しています。
Q. メールはどのくらい保存されますか？
最新の一定件数を短期間保持し、古いメールから順次削除していきます。長期保存が必要なメールは、必ず別途保存してください。
Q. スマートフォンでも使えますか？
はい、スマートフォン・タブレット・PCのどれからでも利用可能です。ブラウザでsutemeado.comを開くだけで、端末やOSに依存せず使えます。

ログイン

API Documentation

Base URL: https://sutemeado.com
Rate Limit: 100 requests/minute
Auth: Password required for mailbox operations (POST body)

GET/api/new-address

新規アドレスを作成

curl https://sutemeado.com/api/new-address

// Response
{
  "success": true,
  "address": "abc123@sutemeado.com",
  "password": "xyz789pass",
  "domain": "sutemeado.com"
}
POST/api/login

ログイン（アドレスとパスワードで認証）

curl -X POST https://sutemeado.com/api/login \
  -H "Content-Type: application/json" \
  -d '{"address":"abc123@sutemeado.com","password":"xyz789pass"}'

// Response
{
  "success": true,
  "address": "abc123@sutemeado.com",
  "mails": [...]
}
POST/api/mailbox/:address

メール一覧を取得

curl -X POST "https://sutemeado.com/api/mailbox/abc123@sutemeado.com" \
  -H "Content-Type: application/json" \
  -d '{"password":"xyz789pass"}'

// Response
{
  "success": true,
  "address": "abc123@sutemeado.com",
  "mails": [
    {
      "id": "mail_xxx",
      "from": "sender@example.com",
      "subject": "Hello",
      "body": "Message text...",
      "html": "<p>HTML version...</p>",
      "receivedAt": 1711234567890
    }
  ]
}
POST/api/mailbox/:address/:mailId

特定のメールを取得

curl -X POST "https://sutemeado.com/api/mailbox/abc123@sutemeado.com/mail_xxx" \
  -H "Content-Type: application/json" \
  -d '{"password":"xyz789pass"}'
DELETE/api/mailbox/:address/:mailId

メールを削除

curl -X DELETE "https://sutemeado.com/api/mailbox/abc123@sutemeado.com/mail_xxx" \
  -H "Content-Type: application/json" \
  -d '{"password":"xyz789pass"}'
DELETE/api/mailbox/:address

全メールを削除

curl -X DELETE "https://sutemeado.com/api/mailbox/abc123@sutemeado.com" \
  -H "Content-Type: application/json" \
  -d '{"password":"xyz789pass"}'
DELETE/api/address/:address

アドレスを削除（全メールも削除されます）

curl -X DELETE "https://sutemeado.com/api/address/abc123@sutemeado.com" \
  -H "Content-Type: application/json" \
  -d '{"password":"xyz789pass"}'
PUT/api/address/:address/password

パスワードを変更

curl -X PUT "https://sutemeado.com/api/address/abc123@sutemeado.com/password" \
  -H "Content-Type: application/json" \
  -d '{"currentPassword":"xyz789pass","newPassword":"newpass123"}'

一時メール・使い捨てメール「Sutemeado」とは？

Sutemeado（捨てメアド）は、登録不要・完全無料で使える一時メールサービスです。スパム防止やプライバシー保護に最適な、使い捨て可能なメールアドレスを即座に発行できます。

登録不要ですぐ使える

メールアドレスとパスワードを入力するだけで、即座に一時メールアドレスを取得できます。個人情報の入力は一切不要です。

完全無料で利用可能

料金は一切かかりません。広告収入で運営しているため、ユーザーは無料で一時メールサービスを利用できます。

30日間の保持期間

受信したメールは30日間保持されます。パスワードを設定することで、アドレスを保護し後から確認することも可能です。

3ステップで簡単に使える一時メール

1

アドレスを作成

トップページで好きなメールアドレスとパスワードを入力して「作成」ボタンをクリック

2

メールアドレスをコピー

生成された一時メールアドレスをコピーして、サービス登録や配信先に入力

3

メールを受信

Sutemeadoで受信メールを確認。パスワードを入力してログインすればいつでも確認可能

安全性・プライバシー保護

  • HTTPS対応：全通信は暗号化されており、安全に利用できます
  • パスワード保護：作成したアドレスにパスワードを設定し、他人からのアクセスを防止
  • 個人情報不要：メールアドレス作成に個人情報は一切不要
  • 自動削除：30日後にメールが自動削除され、プライバシーを保護
  • スパム防止：自分の本メールアドレスを登録先に教える必要がない

一時メール・捨てメアドの活用シーン

スパムメール防止

不明なサイトへの登録時に使い捨てメールを使用し、自分の本メールアドレスを守る

開発・テスト用途

アプリ開発時のメール送信テストに利用。複数アドレスで同時テストも可能

海外サービスの登録

日本未対応の海外サービスやSNS登録時に使用。安全に試せる

キャンペーン・特典受け取り

メルマガ登録が必要なキャンペーン参加時に使用。後から簡単に削除

掲示板・フォーム投稿

匿名性が必要な場面で、実メールアドレスを公開せずにコミュニケーション

Eコマースの会員登録

一度きりの購入時に使用。購入後はメールアドレスごと削除可能

よくある質問（FAQ）

一時メールは無料ですか？

はい、Sutemeadoは完全無料で利用できます。広告収入で運営しているため、ユーザーは無料で一時メールサービスを利用できます。

メールはどれくらい保持されますか？

受信したメールは30日間保持されます。30日経過すると自動的に削除されます。

パスワードを忘れた場合は？

パスワードは復元できません。新しい一時メールアドレスを作成してください。

スマートフォンでも使えますか？

はい、スマートフォン・タブレットからもブラウザでアクセスして利用可能です。専用アプリは不要です。
